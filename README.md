# Stock Analyzer

A beginner-friendly stock analysis application that lets you search any stock, view real market data, and understand charts, metrics, and market context in simple language.

Built with **C++**, **Python**, **Java**, **HTML/CSS/JS**, and **Electron** — five languages working together as one application.

## What It Does

- **Search any stock** by ticker symbol (AAPL, TSLA) or company name (Apple, Tesla)
- **View live price data** including current price, daily change, and market status
- **Interactive price charts** with selectable time ranges (1D, 5D, 1M, 6M, 1Y, 5Y)
- **Volume charts** with color-coded bars showing buying vs selling pressure
- **16 key statistics** — Market Cap, P/E Ratio, Forward P/E, EPS, Volume, Avg Volume, 52-Week High/Low, Beta, Dividend Yield, Price/Book, Profit Margin, Debt/Equity, and more
- **Technical indicators** — Trend direction, RSI, MACD, Volatility, Support/Resistance, SMA 20/50/200, period return
- **Plain-English analysis** — 8 categories of insight generated by the Java interpretation engine covering price movement, 52-week positioning, volume activity, valuation, volatility, market cap classification, dividends, and price-to-book
- **Recent news headlines** with publisher, timestamps, and clickable links
- **Learn page** — 19 stock terms explained with definitions, why they matter, typical ranges, and category filters

### Technology Breakdown

| Layer | Technology | Purpose |
|-------|-----------|---------|
| **Desktop Window** | Electron (Chromium) | Cross-platform native window — works on Linux, WSL2, macOS, Windows |
| **Backend Server** | C++ with cpp-httplib | Serves the UI and handles all API requests on localhost |
| **Analysis Engine** | C++ | Fast technical analysis — SMA, EMA, RSI, MACD, volatility, trend detection, support/resistance |
| **Data Fetching** | Python with yfinance | Pulls real stock data from Yahoo Finance (free, no API key) |
| **News** | Python with yfinance | Recent headlines with publisher and timestamps |
| **Interpretation** | Java | Generates plain-English explanations of stock behavior across 8 categories |
| **Glossary** | Java | 19 stock market terms with definitions, relevance, and typical ranges |
| **Frontend UI** | HTML, CSS, JavaScript | Dark-themed dashboard with search, charts, stats, analysis, news, and learn pages |
| **Charts** | Chart.js | Interactive price and volume charts with hover tooltips |
| **Caching** | C++ (in-memory) | TTL-based cache — quotes (30s), history (1-5min), glossary (1hr) |
| **Data Format** | JSON (nlohmann/json) | All components communicate via JSON over local HTTP |

### Why These Languages?

- **C++** — Speed. The analysis engine processes price history through SMA, EMA, RSI, MACD, and trend detection in milliseconds. The HTTP server handles requests instantly. The cache is thread-safe and fast.
- **Python** — Data access. yfinance is the most reliable free stock data source available. No API key, no trial, no rate limits for normal use.
- **Java** — Text generation. The interpretation engine takes raw numbers and produces 6-8 readable paragraphs explaining what they mean. The glossary engine serves 19 structured educational entries.
- **JavaScript** — Interactivity. Chart.js renders responsive price and volume charts. The search auto-suggests as you type.
- **HTML/CSS** — UI layout and styling. Dark theme, rounded cards, responsive grid, custom scrollbars.

## API Endpoints

The C++ server exposes these REST endpoints on `localhost:8089`:

| Endpoint | Description |
|----------|-------------|
| `GET /api/health` | Health check |
| `GET /api/search?q=apple` | Search tickers by name or symbol |
| `GET /api/quote/AAPL` | Current price, stats, company info |
| `GET /api/history/AAPL?period=1mo` | Historical OHLCV data (1d, 5d, 1mo, 6mo, 1y, 5y) |
| `GET /api/analysis/AAPL?period=1y` | Technical indicators (SMA, EMA, RSI, MACD, trend, volatility) |
| `GET /api/interpret/AAPL` | Plain-English analysis (Java) |
| `GET /api/news/AAPL` | Recent news headlines |
| `GET /api/glossary` | All 19 glossary terms |

## Requirements

### System
- **Linux** (Ubuntu/Debian recommended, WSL2 supported)
- **g++** with C++17 support (usually pre-installed)
- **Python 3.8+** (usually pre-installed)

### Installed by Setup Script (no sudo required)
- **Node.js** — for the Electron desktop window (installed to `~/.local/node/`)
- **JDK 17** — for the Java interpretation engine (installed to `~/.local/jdk/`)
- **pip + yfinance** — for stock data fetching (installed to `~/.local/`)
- **Electron** — for the application window (installed via npm)

## Setup — First Time

### Quick Setup (Recommended)

```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```

This automatically:
1. Installs JDK 17 and Node.js to your home directory (no sudo needed)
2. Bootstraps pip and installs yfinance
3. Compiles the Java modules
4. Builds the C++ backend
5. Installs Electron

### Package as Standalone App

After setup, create a distributable application:

```bash
bash scripts/package.sh
```

This creates `dist/StockAnalyzer-linux-x64/` — a self-contained folder with everything bundled. You can move this folder anywhere.

## How to Use It

### Searching for a Stock
1. Open the app — you'll see a search bar on the home screen
2. Type a ticker symbol like **AAPL** or a company name like **Apple**
3. Press Enter or click Search
4. The stock detail page loads with all the data

### Reading the Stock Page
- **Header** — Company name, ticker, current price, daily change (green/red)
- **Chart** — Click time range buttons (1D, 5D, 1M, 6M, 1Y, 5Y) to see different periods. Volume bars below show trading activity.
- **Key Statistics** — 16 metrics in a card grid
- **Technical Indicators** — Trend, RSI, MACD, Volatility, Support/Resistance, moving averages
- **Analysis** — 6-8 plain-English paragraphs explaining what the data means
- **News** — Recent headlines with publisher and time

### Quick Access
Click any of the popular stock chips on the home page (AAPL, GOOGL, MSFT, TSLA, AMZN, NVDA, META, JPM) for instant access.

### Learning What Metrics Mean
- Click **Learn** in the navigation bar
- Filter by category (Valuation, Profitability, Trading Activity, Risk, etc.)
- Search for specific terms
- Each entry explains: what it is, why it matters, and what values are typical
